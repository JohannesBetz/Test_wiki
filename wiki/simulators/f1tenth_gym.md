# F1TENTH Gym

## Definition

F1TENTH Gym is a lightweight 2D racing simulator with an OpenAI Gym-style interface for [[f1tenth]]-scale autonomous racing.

## Key Approaches

[[autonomous_vehicles_on_the_edge]] presents F1TENTH Gym as a common environment for reinforcement learning, optimization, and small-scale racing algorithm development.
