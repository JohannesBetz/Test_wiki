# Swift Drone Racing Data

## Definition

Swift Drone Racing Data refers to the motion-capture race recordings and analysis code released with [[champion_level_drone_racing_using_deep_reinforcement_learning]].

## Papers Using This

- [[champion_level_drone_racing_using_deep_reinforcement_learning]]

## Notes

The paper reports that motion-capture recordings of the race events and accompanying analysis code are available on Zenodo at DOI `10.5281/zenodo.7955278`. Pseudocode for Swift is also provided there, while the full source code is not publicly released due to misuse concerns.
