# Race Against the Machine Dataset

## Definition

Race Against the Machine Dataset is a fully annotated open-design dataset of autonomous and piloted high-speed flight introduced in [[race_against_the_machine_a_fully_annotated_open_design_dataset_of_autonomous_and_piloted_high_speed_flight]].

## Key Approaches

- Designed for high-speed aerial autonomy rather than generic slow-flight robotics.
- Includes both autonomous and human-piloted flight data.
- Positioned as a public resource for perception, learning, benchmarking, and behavior comparison in aggressive flight regimes.

## Papers Using This

- [[race_against_the_machine_a_fully_annotated_open_design_dataset_of_autonomous_and_piloted_high_speed_flight]]

## Notes

For this vault, the dataset is important because high-speed aerial autonomy often lacks open, richly annotated data collected under realistic aggressive-flight conditions. This paper contributes a public resource directly aimed at that gap.
