## Objective

LeMo aims to improve the mobility of real legged robots through learning-based methods that transfer directly from simulation to physical systems.

According to the CORDIS fact sheet and reporting pages, the project investigates methods by which learned behaviors can move directly from simulator to robot so that legged systems gain more advanced locomotion and navigation capabilities in challenging environments.

The keywords exposed in the grant material make the focus very clear:

- legged robotics,
- locomotion control,
- learning locomotion,
- and machine learning.

So this is not only a grant about one robot platform, but a programmatic investment in the learned locomotion and sim-to-real legged-mobility stack.
