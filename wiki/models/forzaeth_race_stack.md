# ForzaETH Race Stack

## Definition

ForzaETH Race Stack is a modular autonomous racing software stack for scaled 1:10 head-to-head racing using commercial off-the-shelf hardware.

## Key Approaches

- Modular full-stack architecture for perception, state estimation, planning, and control.
- Designed for both time trials and unrestricted head-to-head racing.
- Built for the [[f1tenth]] ecosystem and similar scaled racing platforms.
- Emphasis on reproducibility, ease of deployment, and adaptability to track layout and friction changes.

## Open Problems

Open problems include scaling the same architectural ideas to full-size racecars, preserving reproducibility as stacks become more specialized, and balancing modularity with the performance benefits of tighter end-to-end integration.
