# He From Seeing to Experience Source

> Source: `raw/pdfs/He_From_Seeing_To_Experience.pdf`
> Paper: From Seeing to Experiencing: Scaling Navigation Foundation Models with Reinforcement Learning

## Summary

This source argues that navigation foundation models should not remain purely perception-heavy systems trained only on static visual data. Instead, they should be pushed into `experience` through reinforcement learning so that navigation competence grows from interaction rather than only from seeing.

For this vault, that makes the paper a strong bridge between the `foundation models`, `cognitive navigation`, and `era of experience` branches.

## Method / Framing

The core framing is right in the title:

- start from broad visual or navigation priors,
- then move from `seeing` to `experiencing`,
- and use reinforcement learning to scale capability through interaction.

That is a useful step beyond papers that treat foundation models mainly as static representational backbones. This source instead suggests that embodied competence may require converting those priors into closed-loop experience.

## Notes / Critique

The strength of the source is that it makes the experience transition concrete in a robotics-flavored domain. It is not only a manifesto about experience in the abstract; it is about converting pretrained navigation competence into interaction-shaped competence.

Its limitation, from the vault's perspective, is that the current local source evidence does not yet justify strong claims about embodiment scale, hardware transfer, or real-world deployment maturity. So the page stays at the method-and-framing level.
