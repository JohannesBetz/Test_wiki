# Multiple Robot Embodiments

## Definition

This hardware class is for papers that span several robot bodies or hardware classes rather than demonstrating capability on one primary platform.

## Relationship to the Wiki

This page belongs naturally near [[real_world_robotic_reinforcement_learning]], [[embodied_autonomous_intelligence]], and [[foundation_models_for_intelligent_decision_making]].

It is useful for surveys and cross-domain synthesis papers where the main contribution is not one hardware demonstration, but a comparison across drones, quadrupeds, mobile robots, manipulators, and related platforms.
